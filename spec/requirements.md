# Requirements

> **维护规则**：本文件是**项目整体需求**，仅在**人工明确要求**时修改。AI 不得擅自更新。

---

## 1. 项目定位

<!-- 一句话说清楚：做什么产品、给谁用、解决什么问题 -->

`[v0.1 新增]` 一款个人自用的本地语音克隆桌面应用，基于 dots.tts 模型，用本机 NVIDIA GPU 把任意中文文本变成指定音色的语音。

## 2. 目标用户

<!-- 核心用户画像、使用场景 -->

`[v0.1 新增]` 单一用户：开发者本人（ck）。场景：在自己电脑上克隆自己/家人/授权样本的音色，批量生成中文语音（配音、有声片段、自定义提示音等）。

## 3. 核心功能

<!-- 按优先级列出 MVP 功能清单 -->

- [ ] **R-v0.1-ck-1** `[v0.1 新增]` **单段 continuation 克隆**：输入目标文本 + 选参考音频 + 手输参考转录 → 调 dots.tts continuation 模式生成 → 试听 + 保存 wav
- [ ] **R-v0.1-ck-2** `[v0.1 新增]` **音色库管理**：保存「参考音频 + 转录 + 命名 + 备注」为音色条目，支持添加/删除/重命名/试听/选择
- [ ] **R-v0.1-ck-3** `[v0.1 新增]` **批量合成**：多行文本（一行一段）+ 从音色库选一个音色 → 顺序生成 → 多 wav 按序号命名保存到指定目录，带进度

## 4. 非功能需求

<!-- 性能、合规、安全、多端、国际化等 -->

- **性能** `[v0.1 新增]`：本地 NVIDIA GPU ≥12GB 显存；模型一次加载常驻、多次合成复用
- **合规** `[v0.1 新增]`：个人自用，不做 AI 水印/标注；用户自己对参考音频授权负责
- **安全** `[v0.1 新增]`：纯本地运行，无网络服务、无鉴权、无多用户
- **多端** `[v0.1 新增]`：桌面 App（目标 OS 优先 Windows，待 design 阶段定）

## 5. 非目标（明确不做什么）

<!-- 列出容易被误以为要做但实际不做的事情——防止范围蔓延 -->

`[v0.1 新增]` 以下功能 v0.1 明确不做：

- x-vector-only / random-voice 克隆（仅 continuation）
- ASR 自动转录（参考转录手输）
- 参数调节面板（用默认 num_steps=10, guidance=1.0, language=ZH/none, seed=42）
- 历史记录
- fine-tuning / 流式合成
- 多语言（仅中文，中英混排留待后续）
- mp3 输出（仅 wav）
- AI 水印 / 元数据标注
- 鉴权 / 多用户 / 限流 / 远程服务

## 6. 成功标准

<!-- 用可度量指标回答"怎样算成功" -->

`[v0.1 新增]`：

- 能在本地 GPU 上稳定加载 dots.tts 模型并完成单段克隆
- 音色库条目可保存/复用，连续多次合成不重载模型
- 批量合成 10 段文本能顺序完成，单段失败不中断整体
- 桌面 App 可打包安装并在本人电脑上独立运行
